# Drop SoundFonts here

Tunerize scans this folder for `.sf2` and `.sf3` files at startup. Restart the app after adding files.

## Recommended free SoundFonts

| Name | Style | License | Size | Notes |
|------|-------|---------|------|-------|
| **FluidR3 GM** | Full GM bank | MIT | ~140 MB | The FluidSynth default — solid all-rounder |
| **GeneralUser GS** | Full GM/GS bank | CC BY-SA | ~30 MB | Lightweight, very compatible |
| **NES SoundFont** (search GitHub) | 8-bit / chiptune | varies | ~1 MB | The classic retro-game sound |
| **Roland SC-55** (museum recreations) | 90s GM synth | varies | ~50 MB | The "Final Fantasy / DOOM MIDI" sound |
| **Sega Genesis FM** | YM2612 chip | varies | ~5 MB | Sega 16-bit era |

These files are large and licensed individually — Tunerize ships none of them. Search "free SoundFont GM" or "chiptune sf2" and grab what fits your use case.

## Why isn't this folder pre-populated?

Three reasons:
1. **License variance** — bundling SoundFonts with mixed licenses creates redistribution ambiguity.
2. **Repo size** — even one full GM bank doubles the repo size.
3. **Personal taste** — the 8-bit folks want chiptune, the orchestrators want SC-55, the lo-fi folks want Mellotron. No default fits all.

Use **Browse Online…** in SoundFont mode to search public libraries from inside Tunerize. First-run FluidR3 GM onboarding is still planned.
